# Quality Check & Scoring Reference

## The 4 C's — Editing Checklist

Use for every piece of copy before finalizing:

| C | Question | Check |
|---|----------|-------|
| **Clear** | Is it instantly understood? | No jargon, no ambiguity |
| **Concise** | Does every word earn its place? | Remove filler, tighten sentences |
| **Compelling** | Does it create desire to continue? | Emotion, curiosity, self-interest |
| **Credible** | Is it believable? | Specific, grounded, not hyperbolic |

---

## The 4 U's — Headline Scoring

Score each dimension 1-4. **Total 12+ tends to outperform.**

| U | Question | Scoring |
|---|----------|---------|
| **Urgent** | Why act now? | 1=none, 4=immediate action needed |
| **Unique** | What makes it different? | 1=generic, 4=never seen before |
| **Useful** | How does it help them? | 1=unclear benefit, 4=obvious value |
| **Ultra-specific** | For who exactly? | 1=everyone, 4=specific segment |

Example scoring:
```
"The 3-minute morning routine busy moms use to lose 10lbs by summer"

Urgent: 3 (by summer deadline)
Unique: 3 (3-minute + busy moms angle)
Useful: 4 (weight loss result)
Ultra-specific: 4 (busy moms, 10lbs, timeframe)

Total: 14 ✓ Strong headline
```

## The 4 S's — Quick Hook Check

| S | Check |
|---|-------|
| **Short** | Under 10 words for H1? |
| **Simple** | Understood at first glance? |
| **Specific** | Concrete details, not abstractions? |
| **Surprising** | Pattern interrupt present? |

## Hook Pre-flight Checklist

Before publishing any hook, verify:

- [ ] Evokes emotional reaction in first 0.5 seconds?
- [ ] Sounds like real person, not brand?
- [ ] Contains specific details (numbers, dates, names)?
- [ ] Shows transformation (before → after)?
- [ ] Creates curiosity gap (open loop)?
- [ ] Has authority signal (if needed)?
- [ ] Clear who it's for (target identification)?
- [ ] Works as standalone (no context needed)?
- [ ] Matches audience awareness level?
- [ ] Has implied payoff (not empty clickbait)?

---

## Ad Copy Pre-flight Checklist

- [ ] Matches awareness level of audience?
- [ ] Sells benefit, not feature?
- [ ] Addresses emotional job, not just functional?
- [ ] Social proof appears before pitch?
- [ ] Main objections addressed?
- [ ] One clear CTA?
- [ ] Uses "you" language (direct address)?
- [ ] Active voice, not passive?
- [ ] Short sentences (5-15 words)?
- [ ] No clichés ("game-changer", "revolutionary")?

## UGC Script Checklist

- [ ] Format native to platform?
- [ ] Raw > polished (authentic feel)?
- [ ] One clear benefit (not feature list)?
- [ ] Specific details (not generic claims)?
- [ ] Hook in first 3 seconds?
- [ ] Sounds conversational?
- [ ] Natural CTA (not salesy)?

## CTA Checklist

- [ ] Clear action verb?
- [ ] Low friction (easy next step)?
- [ ] Promises result or benefit?
- [ ] 5 words or fewer?
- [ ] Matches ad tone?
- [ ] No punctuation at end (except !, →)?

## Quick Tests

### The Grandma Test
Would a normal person understand it instantly without context? If not, simplify.

### The "So What?" Chain
Ask "So what?" after each statement until you hit the real benefit. If you can't get to a clear benefit in 3 "so whats", the copy isn't landing.

Example:
```
Statement: "Our app has AI-powered analysis"
So what? → "It processes data automatically"
So what? → "You don't have to do it manually"
So what? → "You save 4 hours every week"
✓ Real benefit found
```

### The Finger Test
Point to each word. Can you delete it without losing meaning? If yes, delete it.

Before: "Our truly amazing product really helps you to easily save money"
After: "Save money effortlessly"

### The Headline Halving Rule
Try cutting the headline in half. Often the shorter version is stronger.

Before: "The Complete Guide to Building Your First Successful Online Business from Scratch"
After: "Build Your First Online Business"

### The Screenshot Test
Would someone screenshot this to share with a friend? If not, the hook isn't strong enough.

## R.A.D.U. Framework (Landing Pages / CRO)

Use for landing page copy evaluation:

| Element | Question | Check |
|---------|----------|-------|
| **Relevance** | Does it match what brought them here? | Message match with ad |
| **Attention** | Does it grab attention immediately? | Strong headline, visual |
| **Distraction** | Is there friction or confusion? | Remove unnecessary elements |
| **Urgency** | Why should they act now? | Scarcity, deadline, FOMO |

## Common Copy Mistakes to Avoid

### Hook Mistakes
| Mistake | Fix |
|---------|-----|
| Too generic | Add specific numbers, details |
| Too long | Cut to 3-7 words |
| No emotion | Add trigger (fear, curiosity, desire) |
| Sounds like an ad | Use conversational language |
| Wrong awareness level | Match to audience stage |

### Body Copy Mistakes
| Mistake | Fix |
|---------|-----|
| Feature-focused | Convert to benefits |
| Too long sentences | Break into 5-15 word chunks |
| Passive voice | Rewrite active |
| No proof | Add testimonial, data, authority |
| Weak CTA | Strengthen with benefit or urgency |

### Story Mistakes
| Mistake | Fix |
|---------|-----|
| No conflict | Add struggle/pain before solution |
| Generic details | Add specific names, numbers, places |
| Weak ending | Write strong kicker line |
| Too long | Cut to 150-250 words |
| Third person | Rewrite first person |

## Quality Tiers

### Tier 1: Minimum Viable (Test-worthy)
- Grammatically correct
- Clear value proposition
- Appropriate length
- Basic emotional appeal

### Tier 2: Competitive (Average performer)
- Specific details
- Social proof included
- Addresses one objection
- Clear CTA

### Tier 3: High-converting (Top 20%)
- Pattern interrupt hook
- Strong emotional trigger
- Multiple proof points
- Urgency element
- Perfect awareness match

### Tier 4: Viral potential (Top 5%)
- Screenshot-worthy hook
- Personal story element
- Contrarian or surprising angle
- Perfect specificity
- Memorable ending

## Rewrite Prompts

When copy needs improvement, use these prompts:

**For hooks:**
"Make this more specific by adding [numbers/timeframes/names]"
"Add a pattern interrupt to the beginning"
"Convert to [question/statement/command] format"
"Apply [trigger name] psychological trigger"

**For body:**
"Convert features to benefits"
"Add social proof before the pitch"
"Address the objection: [objection]"
"Shorten all sentences to under 15 words"

**For CTAs:**
"Add urgency"
"Include the benefit in the CTA"
"Reduce to 3 words maximum"
"Match the ad's emotional tone"
