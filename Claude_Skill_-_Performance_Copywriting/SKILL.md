# Performance Marketing Copywriting Skill

## Core Philosophy

**Bad hooks describe. Good hooks make you FEEL something.**
**Stop selling features. Start selling transformation.**

People don't buy products. They buy progress in their lives and emotional states.

| What marketers think they sell | What people actually buy |
|-------------------------------|-------------------------|
| Gym membership | Hope that this time will be different |
| Project management tool | "Finally everything is under control" feeling |
| Language app | The moment they can order in French without panic |
| Budgeting app | "I'm not failing at adulting" feeling |

## Quick Workflow

### Step 1: Define Inputs (Copy Brief)

Always gather before writing:

```
PRODUCT: [Name]
CATEGORY: [app/SaaS/course/physical/service]
VALUE PROP: [One sentence — what it does]
TRANSFORMATION: [Before → After state]
PRIMARY PAIN: [Main pain point]
SECONDARY PAINS: [3-5 additional]
KEY BENEFITS: [3-5 benefits]
PROOF: [Stats, testimonials, authority]
PRICE/MODEL: [Free/subscription/one-time]
TARGET AUDIENCE: [Primary + secondary segments]
OBJECTIONS: [Typical objections]
AWARENESS LEVEL: [Unaware/Problem/Solution/Product/Most Aware]
PLACEMENT: [Meta primary/headline/LP/email/etc.]
TONE: [UGC/expert/contrarian/friendly/premium]
LANGUAGE: [EN/UA/etc.]
```

### Step 2: Match Awareness Level

| Level | State | Hook Strategy |
|-------|-------|---------------|
| **Unaware** | Don't know they have a problem | Problem revelation, story, intrigue |
| **Problem Aware** | Feel pain, no solution | Pain agitation, empathy |
| **Solution Aware** | Know solutions exist, not your product | Differentiation, mechanism |
| **Product Aware** | Know product, not convinced | Proof, objection handling |
| **Most Aware** | Ready to buy | Offer, urgency, easy CTA |

### Step 3: Select Framework

**Short-form (Meta ads, hooks):**
- PAS (Problem → Agitate → Solution)
- BAB (Before → After → Bridge)
- Hook-Story-Offer

**Long-form (sales pages, emails):**
- PASTOR (Problem → Amplify → Story → Transformation → Offer → Response)
- AIDA (Attention → Interest → Desire → Action)
- Perry Belcher's 21-Step

## Hook Structure Quick Reference

### Psychological Elements (Every Hook Needs)

1. **Pattern Interrupt** — stops the scroll
2. **Emotional Trigger** — evokes emotion in 0.5 seconds
3. **Self-Identification** — reader recognizes themselves
4. **Open Loop** — creates desire to know more
5. **Specificity** — concrete details, not abstractions

### 15 Hook Types (Mix for Testing)

1. Confession ("I used to...")
2. Contrarian (Challenges common belief)
3. Question (Provocative)
4. Statement (Bold claim)
5. Data/Number (Starts with figure)
6. Command (Direct order)
7. Social Proof (References others)
8. Fear/Warning (Alert)
9. Curiosity Gap (Intrigue)
10. Identity (Addresses who they are)
11. Time-based (Deadline/timeframe)
12. Comparison (A vs B)
13. Result (Outcome focus)
14. Problem-Agitate (Intensifies pain)
15. Permission (Gives permission to want)

### Core Hook Formulas

```
[NUMBER] + [TIMEFRAME] + [RESULT]
→ "52 BOOKS A YEAR WITHOUT ADDING HOURS"

[I/YOU] + [PAST TENSE] + [PAIN POINT]
→ "I USED TO FEEL GUILTY ABOUT NOT READING"

[STOP/START] + [CURRENT BEHAVIOR]
→ "STOP PAYING FOR AUDIOBOOKS"

[YOUR] + [ASSET] + [NEGATIVE STATE]
→ "YOUR BOOK LIST KEPT GROWING — BUT YOU NEVER READ"

[YOU'RE NOT] + [NEGATIVE] + [YOU'RE] + [REFRAME]
→ "YOU'RE NOT LAZY. YOU'RE OVERLOADED."

[BIG NUMBER] + [CONTEXT]
→ "2,080 HOURS — THAT'S YOUR ANNUAL COMMUTE"
```

## Body Copy Structure

```
1. PROBLEM AMPLIFICATION (1-2 sentences)
   → Intensifies pain or problem

2. SOLUTION INTRODUCTION (1-2 sentences)
   → Presents the solution

3. MECHANISM/HOW (1-2 sentences)
   → Explains how it works

4. BENEFIT/RESULT (1-2 sentences)
   → Shows the outcome

5. PROOF (optional, 1 sentence)
   → Social proof or statistic
```

### Writing Rules

✓ Active voice, not passive
✓ Second person (YOU/YOUR) — direct address
✓ Specificity over abstractions
✓ Strong verbs over weak ones
✓ Short sentences (5-15 words)
✓ Conversational tone, not academic
✓ Emotional, not rational

✗ Avoid: "very", "really", "just", "actually", "basically"
✗ Avoid: Passive constructions
✗ Avoid: Complex sentences with multiple clauses
✗ Avoid: Jargon without explanation
✗ Avoid: Clichés ("game-changer", "revolutionary")

Format: 2-5 words, ALL CAPS or Title Case, no period.

## Story Structure (For Poster Ads / Long-form)

```
LENGTH: 150-250 words
POV: First person (I/My/Me)
TONE: Emotional, authentic, vulnerable

ARC:
1. SETUP (2-3 sentences) — context + character
2. PROBLEM/PAIN (3-4 sentences) — struggle + impact
3. DISCOVERY MOMENT (1-2 sentences) — finding the solution
4. TRANSFORMATION (3-4 sentences) — how it changed things
5. RESULT/NEW REALITY (2-3 sentences) — emotional payoff
6. KICKER (1 sentence) — strong ending
```

## Testing Hierarchy

Test in this order (highest to lowest impact):

1. **HOOK** (60-70% of success) — First 3 seconds / 125 chars
2. **ANGLE** (20-25%) — Pain vs Aspiration, Logic vs Emotion
3. **FORMAT** (10-15%) — Static vs Video, UGC vs Produced
4. **OFFER** (5-10%) — Discount vs Bonus, Trial vs Guarantee

## Critical Reminders

1. **Variety > Perfection** — 10 different approaches beat 10 variations of one
2. **Emotion first, logic second** — People buy emotionally, justify logically
3. **Specificity sells** — "47 books" > "many books"
4. **Test extremes** — Include both soft and aggressive variants
5. **Write for skimmers** — Most won't read everything. H1 must work alone
6. **Every word earns its place** — If a word can be deleted without losing meaning, delete it
7. **Hook in first 3 seconds** — That's all the time you have
8. **Match awareness level** — Wrong level = poor conversion
