# Mass Copy Generation Reference

## Canonical Output: 11 Categories × 10 Variations

When generating copy variations at scale, produce 11 categories with 10 variations each (110 total texts per creative concept).

---

## Category 1: NEW HOOKS
**Purpose:** Alternative H1+H2 with same core message
**Output format:** H1 + H2

Generate 10 hooks varying:
- Hook type (question, statement, confession, etc.)
- Psychological trigger
- Specificity level
- Length (short vs medium)
- Tone (aggressive vs soft)

---

## Category 2: IMPROVED FULL COPY
**Purpose:** Complete ad blocks with different approaches
**Output format:** H1 + H2 + Body + CTA

Generate 10 full copies varying:
- Lead approach (pain vs aspiration)
- Body structure (mechanism vs proof vs story)
- CTA type (benefit vs urgency vs identity)

---

## Category 3: POSTER STORIES
**Purpose:** Long emotional first-person narratives
**Output format:** H1 + Story (150-250 words)

Generate 10 stories using:
- Different character archetypes
- Different emotional arcs (redemption, vindication, discovery, etc.)
- Different specific details and settings
- Different kicker endings

---

## Category 4: SUBHEAD/BODY COMBINATIONS
**Purpose:** Different pairings for same hook
**Output format:** H1 + Subhead + Body

Generate 10 combos varying:
- Subhead type (mechanism, benefit stack, continuation, objection preempt)
- Body focus (problem, solution, proof, transformation)

---

## Category 5: AUDIENCE VERSIONS
**Purpose:** Adaptations for different segments
**Output format:** Audience + H1 + H2 + Body

Required audiences to cover:

| Category | Segments |
|----------|----------|
| Life Stage | Students, New Parents, Retirees, Career Changers |
| Profession | Executives, Healthcare, Legal, Tech, Sales, Teachers |
| Lifestyle | Commuters, Fitness Enthusiasts, Remote Workers, Travelers |
| Challenges | Dyslexia, Vision Issues, Chronic Illness, ADHD, ESL |
| Demographics | Parents, Singles, Caregivers, Multiple Job Holders |

For each audience adapt:
- Pain point (specific to this group)
- Language (terms and slang they use)
- Situation (typical moments in their day)
- Proof (relevant examples)
- Aspiration (what this group wants to achieve)

---

## Category 6: SITUATIONAL VERSIONS
**Purpose:** Tie to specific moments/contexts
**Output format:** Situation + H1 + H2 + Body

Required situations:

| Category | Specific Moments |
|----------|-----------------|
| Daily Routine | Morning Commute, Gym Session, Cooking Dinner, Walking Dog, Laundry |
| Waiting | In Line, At Doctor, In Traffic, For Flight, During Loading |
| Time-Based | Sunday Evening, New Year, End of Quarter, Before Vacation |
| Life Events | New Job, Promotion, Performance Review, Birthday, After Failure |
| Emotional States | Feeling Behind, After Procrastinating, When Overwhelmed, Late at Night |

Situational hook formulas:
```
[TIME/PLACE STAMP] + [RELATABLE MOMENT]
→ "[6:47 AM. STUCK IN TRAFFIC.]"

[YOU'RE DOING X] + [YOU COULD BE DOING Y]
→ "YOU'RE FOLDING LAUNDRY. YOU COULD BE LEARNING."

[SITUATION] + [EMOTION/REALIZATION]
→ "IT'S SUNDAY NIGHT. YOUR READING LIST HASN'T MOVED."
```

**Output format:** Trigger + H1 + H2 + Body

Required triggers to cover:

| Trigger | Keywords | How to Apply |
|---------|----------|--------------|
| FOMO | "Everyone else is...", "You're falling behind" | Others getting what you're not |
| Guilt | "That stack of...", "You said you would..." | Unfulfilled promises to self |
| Shame | "Hiding your...", "Embarrassed by..." | What they hide from others |
| Status | "The top 1%...", "Elite performers..." | Desire to be better |
| Time Scarcity | "No time to...", "Too busy to..." | Lack of time |
| Money Saving | "$X/year wasted", "Free alternative" | Financial benefit |
| Self-Improvement | "Become better...", "Level up..." | Growth desire |
| Social Proof | "2M+ people...", "Join thousands who..." | Others already doing it |
| Exclusivity | "Secret hack...", "They don't share..." | Hidden knowledge |
| Simplicity | "Just one click...", "3 seconds to..." | Ease of solution |
| Future Self | "A year from now...", "Future you..." | Future projection |
| Relief | "Finally...", "No more..." | End of suffering |
| Curiosity | "How do they...", "The truth about..." | Desire to know |
| Belonging | "Join the movement...", "Be part of..." | Group membership |
| Vindication | "Prove them wrong...", "Show them..." | Proving critics wrong |

## Category 8: OBJECTION HANDLING
**Purpose:** Address typical objections head-on
**Output format:** Objection + H1 + H2 + Body

Required objection categories:

| Category | Typical Objections |
|----------|-------------------|
| Quality | "It sounds robotic", "Not as good as real...", "Free means low quality" |
| Effort | "It's probably complicated", "I don't have time to learn", "Too many steps" |
| Value | "I don't need this", "I already have...", "There's a catch" |
| Identity | "This isn't for me", "Real X do Y", "I should be able to..." |
| Trust | "How do I know it works?", "What's the catch?", "Never heard of it" |
| Timing | "I'll do it later", "Not the right time", "Maybe next month" |
| Comparison | "X is better", "I prefer the old way", "My current solution works" |

Objection handling formula:
```
[ACKNOWLEDGE OBJECTION] → [REFRAME] → [NEW PERSPECTIVE]

"AI voices sound robotic"
→ "FORGET EVERYTHING YOU KNOW ABOUT AI VOICES"
→ "Peech sounds so human, you'll forget it's not."
```

## Category 9: CONTRAST PAIRS (BEFORE/AFTER)
**Purpose:** Before/After transformation structures
**Output format:** H1 + Before + After + Body

Structure:
```
BEFORE: [Pain state in present tense]
AFTER: [Gain state in present tense]
BODY: [Bridge between states + product role]
```

Contrast patterns:

| Before | After |
|--------|-------|
| Wasting [resource] | Using [resource] productively |
| Feeling [negative emotion] | Feeling [positive emotion] |
| Failing at [activity] | Succeeding at [activity] |
| [Negative number/metric] | [Positive number/metric] |
| Hiding [thing] | Proud of [thing] |
| Avoiding [situation] | Seeking [situation] |

---

## Category 10: QUESTION VS STATEMENT
**Purpose:** Same message in two formats
**Output format:** Question H1 + Statement H1 + Body

Principle: One core message delivered as:
1. Question — provokes thought, softer approach
2. Statement — direct, confident, authoritative

Formulas:
```
QUESTION: "WHAT IF [BENEFIT] WAS POSSIBLE?"
STATEMENT: "[BENEFIT] IS POSSIBLE."

QUESTION: "WHY ARE YOU STILL [PAIN BEHAVIOR]?"
STATEMENT: "STOP [PAIN BEHAVIOR]."

QUESTION: "TIRED OF [PROBLEM]?"
STATEMENT: "[PROBLEM] ENDS TODAY."
```

---

## Category 11: DATA-DRIVEN HOOKS
**Purpose:** Lead with specific numbers/data
**Output format:** H1 + H2 + Body

Data types for hooks:

| Type | Example Usage |
|------|--------------|
| Time | Hours wasted, time saved, duration |
| Money | Cost comparison, savings, ROI |
| Quantity | Books read, articles consumed, users |
| Percentage | Improvement rates, failure rates |
| Comparison | Speed difference, price difference |
| Social | Number of users, ratings, reviews |
| Personal | Items in backlog, unused purchases |

Data hook formulas:
```
[BIG NUMBER] + [SURPRISING CONTEXT]
→ "2,080 HOURS — THAT'S YOUR ANNUAL COMMUTE"

[NUMBER A] vs [NUMBER B]
→ "250 WORDS/MIN READING. 450 WORDS/MIN LISTENING."

[PERCENTAGE] + [OF WHAT]
→ "91% OF SAVED ARTICLES ARE NEVER READ"

[$AMOUNT] + [WASTE/SAVE FRAME]
→ "$180/YEAR ON AUDIOBOOKS IS INSANE"

[NUMBER] + [TIMEFRAME] + [RESULT]
→ "52 BOOKS A YEAR WITHOUT ADDING HOURS"
```

## Generation Principles

1. **Variety > Perfection** — 10 different approaches beat 10 variations of one
2. **Test extremes** — Include both soft and aggressive variants
3. **Respect the original** — If creative already works, preserve its core insight
4. **Each text is a hypothesis** — Don't try to predict winner, create maximum diversity
5. **Emotion first, logic second** — People buy emotionally, justify logically
6. **Specificity sells** — "47 books" > "many books"
7. **Every word earns its place** — If deletable without losing meaning, delete

## Placement Labels

Always label each variant:
- `[Meta-PT]` = Meta Primary Text (≤125 chars)
- `[Meta-H]` = Meta Headline (≤40 chars)
- `[Other]` = Other placements (LP, email, longer allowed)

Distribution per concept:
- 6 variants optimized for Meta Primary Text
- 2 variants optimized for Meta Headline
- 2 variants optimized for Other placements
